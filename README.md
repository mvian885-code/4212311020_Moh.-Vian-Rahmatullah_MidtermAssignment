Analisis :
Asesment ini meminta pembuatan program untuk klasifikasi huruf tulisan tangan (A–Z) menggunakan Dataset: EMNIST Letters (26 kelas, total 145.600 sampel), Dengan metode Ekstraksi fitur menggunakan HOG (Histogram of Oriented Gradients) dan Klasifikasi menggunakan SVM (Support Vector Machine).
Untuk evaluasi Menggunakan metode Leave-One-Out Cross Validation (LOOCV) dan Menampilkan metrik evaluasi seperti confusion matrix, akurasi, presisi, dan F1-score.
Tujuan utama agar dapat memperoleh performa terbaik melalui tuning parameter HOG dan SVM.

Kesimpulan asesment ini bertujuan untuk mengukur kemampuan mahasiswa dalam mengimplementasikan machine learning untuk klasifikasi citra tulisan tangan menggunakan HOG Feature Extraction dan Support Vector Machine (SVM) dengan dataset EMNIST Letters.
1. Memahami penerapan praktis HOG dan SVM dalam pengenalan karakter tulisan tangan
2. Mampu melakukan eksperimen sistematis (tuning dan validasi) pada model machine learning.
3. Melakukan evaluasi performa model dengan metode Leave-One-Out Cross-Validation (LOOCV), serta menampilkan confusion matrix, accuracy, precision, dan F1-score.
4. Menguasai analisis performa model klasifikasi berbasis data citra dan menunjukkan kemampuan komunikasi ilmiah melalui dokumentasi dan presentasi video.
